常用cli指令

生成没有测试文件的项目:ng new app1 --skip-tests
生成有路由模块的项目:ng new app1 --routing
使用提前编译构建:ng build --aot
最小化打包:ng build --prod
指定模板在ts文件中:ng g c componentName --inline-template
指定不生成spec文件:ng g c componentName --spec=false
将当前应用程序更新为最新版本:ng update --all
