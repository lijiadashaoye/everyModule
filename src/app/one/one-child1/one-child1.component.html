ng-template

ng-template属于后加载的，作为行元素放在当前组件，不与先加载 的一起构成DOM结构，若不指定 ngTemplateOutlet，是不会显示出来的。
若是用createEmbeddedView() 动态创建的，会作为当前整个页面的子元素append进去

用createEmbeddedView() 动态创建ng-template，点击按钮看页面最底下

ViewChildren

将多个组件B、C放到一个组件A内，形成组件组， 然后将A组件放到这个组件组应该安放的位置

这是组件组最终应该放的位置

ContentChildren

将组件A、B一起放到最终位置，但A标签内放入B标签，同时在A组件的html内，用 ng-content表明B组件应该显示的位置

管道参数
管道可以接收任意数量的参数，使用方式是在管道名称后面添加“ : ”和参数值。
如 number: '1.4-4' ，若需要传递多个参数则参数之间用冒号隔开，具体示例如下：

{{ 'semlinker' | slice:0:3 }}

{{ 'semlinker' | slice:0:3 }}

管道链
我们可以将多个管道连接在一起，组成管道链对数据进行处理。

{{ 'semlinker' | slice:0:3 | uppercase }}

{{ 'semlinker' | slice:0:3 | uppercase }}
