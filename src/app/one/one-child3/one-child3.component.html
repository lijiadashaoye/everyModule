vscode 设置

1、先按ctrl(mac是command键) + k，再按ctrl(mac是command键) + 0，可以实现vscode代码折叠

2、按ctrl(mac是command键) + 数学减号'-'，可实现vscode字体变小，数学加号'+'，可实现vscode字体变大

3、每5分钟自动保存一次："files.autoSaveDelay": 30000,

4、控制编辑器是否应在键入后自动设置行格式化："editor.formatOnType": false,Alt+Shift+F实现手动格式化代码

5、调整编辑区域的缩放级别："window.zoomLevel": 0.5,

6、同步 Git 存储库前进行确认（左下角）："git.confirmSync": false,

7、编辑区域字体大小设置："editor.fontSize": 16,

8、重命名或移动文件时自动更新 import 语句的路径："typescript.updateImportsOnFileMove.enabled": "always",

9、控制是否显示工作台底部状态栏中的 Twitter 反馈 (笑脸图标)："workbench.statusBar.feedback.visible": false,

10、限制小地图的宽度："editor.minimap.maxColumn": 30,

11、限制小地图的位置"editor.minimap.side": "left",

12、按ctrl(mac是command键) + 数学减号'-'，可实现vscode字体变小，数学加号'+'，可实现vscode字体变大
